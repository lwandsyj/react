1. react 一切就是组件。

将UI封装成不同的组件（例如常用的UI库）。

功能页面是由多个小组件组成的，比如管理系统类的----》列表组件，查询组件，创建表单组件，等等。

优点：

+ 可组合型，任意组合
+ 可重用性，比如UI库
+ 可维护性


2. 通信
   
+ 系统级别---》 不同系统
  
        iframe 等
        cookie 二级域名
+ 页面级别---》不同页面
  
        传参
        sessionStorage
        localStorage
        cookie

+ 组件之间----》同页面级别
  
    + 父子组件
    + 兄弟组件

3. react 中同页面之间的组件传值
   
   组件间的通信可以理解为组件之间的相互传递数据

+ 父组件向子组件传值，把父组件里面的数据传递给子组件
  
  react 组件和html 元素一样，通过属性（props) 传递给子组件。

+ 子组件传值给父组件，通过事件回调，就想html botton 元素的click 事件等。
+ 兄弟组件（在同一个父元素下同级别）
  
  例如：
            
            <div>
                <p></p>
                <a></a>
            <div>
            a 和 p 是兄弟组件。
